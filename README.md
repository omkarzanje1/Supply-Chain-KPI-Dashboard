# Supply-Chain-KPI-Dashboard

A Power BI dashboard designed to analyze key supply chain performance metrics including inventory management, order fulfillment, delivery performance, and sales trends.

---

## Project Overview

This dashboard helps businesses:

Monitor Inventory Levels

Analyze Order Fulfillment Performance

Track On-Time Delivery %

Measure Revenue and Sales Trends

Identify Top Performing Products

Improve Supply Chain Decision Making

---

## Tools Used

- Power BI
- SQL
- Excel
- Data Modeling
- DAX
- Power Query

---

## Key KPIs

- Total Revenue
- Total Orders
- Inventory Turnover
- Order Fulfillment Rate
- On-Time Delivery %
- Average Delivery Time
